# Changelog

## v1.0.1 — 2026-07-21

### Added

- Draggable on-screen `SA` button for opening and closing Snapjaw Architect
- Persistent launcher position saved between sessions


## v1.0.0 — 2026-07-21

Initial stable release.

### Features

- Searchable NPC catalogue generated from `creature_template`
- Searchable GameObject catalogue generated from `gameobject_template`
- Fast, flexible multi-term catalogue search
- Support for separated and joined names such as `Night Elf` and `NightElf`
- Persistent selected-entry highlighting
- Scrollable result lists
- NPC spawning, information, deletion, and model changes
- GameObject spawning, selection, information, deletion, movement, and rotation
- Automatic Lua catalogue generation from the Snapjaw world database
- Server-side validation and Windows-ready ZIP packaging
- No C++ core modifications or server module required
